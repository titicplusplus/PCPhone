# Installation

You need to install:

- xdotool
- Qt
- https://github.com/nlohmann/json	( apt install nlohmann-json-dev )

```
git clone https://github.com/titicplusplus/PCPhone.git
cd PCPhone/
make
cd extension/button/
make
cd ../../
./ui
```
